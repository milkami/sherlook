    1. POSITION
    2. SPECIALIZATION
    3. COUNTRY
POSITION
CLICK ON SEARCH BUTTON
  • When Search Categories Have Been Selected, Click On SEARCH Button On Bottom Right Of The Page To See Search List Of Available People
FILTER SEARCH LIST
  • Filter Search List By Pressing On One Of The Categories. List Will Be Then Rearranged By The Order You Selected (Words - Alphabatic, Numbers - Highest To Lowest/Lowest To Highest)

POSITION

SPECIALIZATION

COUNTRY

AGE
Age Of The Student.
EXPERIENCE
Experience Based On Sherlooks Own Experience Scoring Formula.
STATUS
Current Status Of Actions That Have Been Done With Student.
SELECT A STUDENT
  • Select A Student / Candidate That Fits Your Requirements Best. By Clicking On The Candidate You Will Be Able To See Full Description And More Information About Them
SAVE SELECTED STUDENT
  • To Save Student In Your Library Click On SAVE Button. When A Student Is In Your Library You Will Be Able To Make A Connection
  • To Gather More Information And Better Description Click On DETAILED EXPERIENCE AND COMPETENCES
  • Personal Information Will Only Be Available After Connection With The Student Has Been Made
